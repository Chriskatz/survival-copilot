Narration audio for slides.html (auto-played per slide).

Files: s1.wav ... s10.wav  (one per slide, in order)
slides.html plays sN.wav for slide N when you press "p" / the Play button.
If a file is missing, it falls back to the browser's built-in voice.

GENERATED BY QVAC's on-device TTS (@qvac/sdk, supertonic):
  node tools/gen_tts.mjs        # reads EN blocks from ../../narration.md
To change voice/speed, edit modelConfig in tools/gen_tts.mjs (voice "F1",
ttsSpeed, ttsNumInferenceSteps) and re-run.

Prefer an even more natural voice? Replace any sN.wav with your own recording
or a high-quality TTS clip (e.g. ElevenLabs) — keep the s1..s10 names.
